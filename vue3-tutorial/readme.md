# Vue Firebase

- npm 전역 설치

```command
npm uninstall -g vue-cli
npm install -g @vue/cli
```

- yarn 전역 설치

```command
yarn global add @vue/cli
```

```command
vue create modal-project
cd modal-project
npm run serve
```
